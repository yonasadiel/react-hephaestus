## Props

This component accepts all input props, with additional props:

| Prop names                      | Description                                       | Default                |
| ------------------------------- | ------------------------------------------------- | ---------------------- |
| <code>placeholder</code>        | label and placeholder for the input               | <code>''</code>        |
| <code>wrapperStyle</code>       | react style for the wrapper                       | <code>'{}'</code>      |
| <code>underlineAnimation</code> | <code>'center' &#124; 'left' &#124; 'none'</code> | <code>'center'</code>  |
| <code>helpText</code>           | text under the input                              | <code>undefined</code> |
| <code>errorText</code>          | error text under the input                        | <code>undefined</code> |
